# Documentation

This directory keeps the course documents and cleanup notes for the lab
repository.

## Course PDFs

- `cpu_manual.pdf`: CPU/lab manual.
- `toolchain.pdf`: toolchain setup guide.
- `introductory_task.pdf`: original introductory task handout.

## Notes

- `setup.md`: how to prepare the local toolchain and hardware access.
- `repository-map.md`: how the final repository is organized and where the
  integrated material came from.
- `branches.md`: summary of the remote branches found in the copied Git repo.

## Archive

- `archive/professor-frink-merge_cache_m_extension.zip`: original archive found
  in the course folder. The extracted source content has been integrated into
  the root `asm/` and `rtl/` directories, excluding the large local toolchain
  folder.

Large extracted toolchains such as `install/` are local environment material,
not source code. They are documented here but intentionally ignored by Git.
